# Bella Vista Restaurant Website

Bella Vista is a modern and responsive restaurant website designed to create an attractive online presence for a restaurant business.

## Features
- Elegant Home Page
- Restaurant Menu Section
- About Us Page
- Contact Form
- Responsive Design
- Smooth Navigation 

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was created to showcase front-end web development skills by building a visually appealing and user-friendly restaurant website.
